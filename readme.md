# Store App

Мобільний застосунок, розроблений на **React Native + Expo** з використанням сучасного стеку та feature-based архітектури.

---

## Запуск проєкту

### 1. Встановлення залежностей

```bash
npm install
```

або

```bash
yarn install
```

---

### 2. Запуск у режимі розробки

```bash
npm run start
```

Після запуску застосунок можна відкрити через:

- **Expo Go** (iOS / Android)
- **Android Emulator**
- **iOS Simulator (macOS)**

---

## Архітектура

У проєкті використовується **feature-based architecture** — підхід, при якому код організований за бізнес-функціональністю, а не за типами файлів.

### Основна ідея

Кожна feature є ізольованим модулем та містить власну:

- UI-логіку
- state management
- API-запити
- типи та моделі
- допоміжні утиліти

Такий підхід спрощує масштабування застосунку та підтримку коду.

---

### Приклад структури проєкту

```
src/
├── app/                # entry point, providers
├── navigation/         # React Navigation
├── features/           # бізнес-модулі
│   ├── auth/

│   │   ├── store/
│   │   ├── api/
│   │   └── types/
│   ├── cart/
│   ├── products/
│   └── profile/
```

## Архітектурні рішення

### Feature-based architecture

- масштабованість
- чітка структура
- зручна командна розробка
- легке додавання нових features

---

### Zustand

Використовується як глобальний state manager.

**Причини вибору:**

- відсутність reducers та actions
- проста async-логіка
- часткове оновлення store
- хороша продуктивність

---

### React Hook Form

Використовується для керування формами:

- мінімальні re-render
- простий API
- легка інтеграція з Yup
- підтримка TypeScript

---

### Yup (валідація)

- централізовані validation schemas
- повторне використання правил
- чисте розділення UI та бізнес-логіки
- повна типізація

---

### React Navigation

- stack / tab навігація
- модульна структура навігації
- підтримка deep linking
- повна сумісність з Expo

---

### Expo

- швидкий старт без native-конфігурації
- Expo Go для тестування
- доступ до native API
- стабільна екосистема
